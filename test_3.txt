ออกแบบการทำงานอย่างไร
  - แบ่งช่วงเวลาสำหรับทำงานและเวลาส่วนตัวให้ชัดเจน
  - ประเมินว่าแต่ละฟังก์ชันควรใช้เวลาในการพัฒนาเท่าไรและเผื่อเวลาสำหรับแก้ไข

รายงานความคืบหน้า
  - อัพโหลดโค้ดผ่าน Github
  - มีการนัดประชุมกับ poject manager
  
เมื่อติดปัญหา
  - ค้นคว้าทางอินเทอร์เน็ต 
  - สอบถามเพื่อนร่วมงาน
  
เมื่องานล้าช้า
  - วางแผนใหม่และปรับเวลาให้แต่ละงานเสร็จเร็วขึ้น 
  - ปรึกษากับ project manager
